# VYUD AI Agent

**VYUD Sales Engagement** — это AI-агент для гипер-персонализации в аутриче.  
Проект направлен на сбор данных о потенциальных клиентах (Account Research, найденные технологии, CRM, активности конкурентов в LinkedIn) и автоматическую генерацию персонализированных писем.

## Проблема
Классический холодный аутрич работает плохо из-за отсутствия контекста. Менеджеры тратят часы на ручной поиск "зацепок" для каждого лида, что не позволяет масштабировать процесс.

## Решение (Гибридная система)
AI-агент, который самостоятельно парсит сайты и соцсети лидов, извлекает специфическую информацию и генерирует уникальные письма. В связке с Google Doc (нашим "Strategic Playbook" для бизнес-расчетов) этот репозиторий представляет собой "Proof of Work" (Техническую витрину).

## Архитектура и Стек
- **Python**: Основной язык логики и скриптов.
- **Pydantic**: Строгая валидация типов и структурированный вывод из LLM. Индустриальный стандарт для надежности извлечения JSON.
- **OpenAI GPT-4o-mini / GPT-4o**: LLM-бэкенд для анализа и генерации текста.

## Состав репозитория
- `smart_parse.py` — Пример парсера сырых LLM-ответов с автоматическим извлечением JSON и валидацией через `Pydantic`.
- В будущем: Скрипты для скрапинга и библиотека системных промптов (Prompt Library).
